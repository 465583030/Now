# Now
一款Android图文精选app，通过抓取网页获得图文列表。目前包含站酷（Zcool）精选、国家地理（National Geographic）每日一图、知乎日报、豆瓣一刻（Moment）,详情页用webview显示，带QQ、微信等分享方式。
视图基于 https://github.com/florent37/MaterialViewPager
